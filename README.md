## Git 제외파일
### node_module 업데이트 제외  >  .gitignore 파일생성후 제외폴더 작성하면 상태가 히든됨
```
node_modules
```



### 기존 리포지토리 깔끔하게 pull / push
```
git pull
git add .
git commit -m "clean push"
git push
```

### 기존 리포지토리 remote 제거
```
git remote remove origin
```

### 새 리포지토리 remote 추가
```
git remote add origin https://github.com/계정/리포지토리
```
